# Run each server (different commands for running each)

Prerequisites:

- Node.js and a package manager (npm, yarn or pnpm).

Frontend (frontendchallengeserver)

Install

```bash
cd frontendchallengeserver
# npm
npm install

```

![alt text](<Screenshot 2025-12-11 at 7.07.02 PM.png>)

Start

```bash
# npm
cd frontendchallengeserver && npm start

```

Avantos (avantos)

Install

```bash
cd avantos
# npm
npm install

```

Start

```bash
cd avantos && npm run dev

```
